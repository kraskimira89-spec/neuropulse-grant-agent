# Дашборд проекта

Веб-дашборд для обзора гранта НейроПульс.

## Запуск

Из корня проекта (с активированным `.venv`):

```bash
streamlit run src/dashboard/app.py
```

По умолчанию откроется на порту 8501. Если чат уже запущен на 8501, укажите другой порт:

```bash
streamlit run src/dashboard/app.py --server.port 8502
```

## Блоки

| Блок | Описание |
|------|----------|
| **Общая информация по гранту** | Шапка: название, сроки, текущий этап, ответственный. Данные из `data/grant_project_dashboard.json` → header. |
| **Прогресс по этапам** | Таблица этапов и мероприятий (план/факт даты, статус, %). Статусы: ⚪ Не начато, 🟡 В работе, 🟢 Выполнено, 🔴 Просрочено. Данные → stages. |
| **Финансовый мониторинг** | Бюджет проекта: план/факт, прогресс-бар освоения, статьи расходов. Данные → budget. |
| **Результаты с участниками** | Целевые показатели (участники, сеансы БАК, эффективность). Данные → indicators. |
| **Информационная активность** | Охват кампании, публикации в СМИ, круглый стол, НейроФест. Данные → info_activity. |
| **Диалог с агентом** | Окно чата в **правой колонке**. При нажатии на кнопки «Быстрые действия по гранту» формулировка (prompt из config) отправляется агенту и ответ показывается в этом же диалоге. |
| **Ближайшие сроки по гранту** | События из **Яндекс Календаря** (CalDAV) или из локального `data/grant_calendar.json`. В настройках: источник (Яндекс / локальный файл), период (дней), показ описания, путь к локальному календарю. |
| **Аудит чата** | Сводка по `data/chat_audit.jsonl`: запросы за период, группировка по дням/сессиям, лимит записей. |
| **Статус агента и API** | Проект, агент, модель, проверка подключения к Yandex AI Studio и опционально Vector Store. |
| **Быстрые действия** | Список кнопок из `grant_tools` в config; настройки — сортировка по id/названию. |
| **База знаний (Vector Store)** | Индекс из конфига, список файлов. Настройки: только completed, лимит файлов. |
| **Ссылки и переходы** | Кнопки «Открыть чат» и «Документация»; URL задаются в настройках. |

У каждого блока в правом верхнем углу — кнопка **⚙️**: по нажатию открываются настройки этого блока.

**Мониторинг гранта:** данные берутся из `data/grant_project_dashboard.json`. Скопируйте `data/grant_project_dashboard.example.json` в `grant_project_dashboard.json` и заполните/обновляйте по мере реализации проекта. Файл `grant_project_dashboard.json` добавлен в `.gitignore`, чтобы фактические цифры не попадали в репозиторий.
