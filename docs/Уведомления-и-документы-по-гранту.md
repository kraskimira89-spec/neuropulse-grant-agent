# Уведомления за 3 дня и генерация документов по гранту

За **3 дня до события** из календаря гранта на почту отправляется напоминание с предложением сформировать документ. Пользователь в чате подтверждает — система создаёт документ Word, отправляет его на почту и при возможности добавляет в базу знаний агента (Vector Store).

---

## Что нужно настроить

### 1. Календарь гранта

Файл **data/grant_calendar.json** (скопируйте из **data/grant_calendar.example.json**). В каждом событии можно указать:

- **date** — дата (YYYY-MM-DD)
- **title** — название (по нему ищется событие в чате)
- **description** — описание
- **document_type** (необязательно): `report` | `event_scenario` | `algorithm` — тип генерируемого документа (по умолчанию отчёт)

### 2. Почта (уведомления и отправка документов)

В **config/.env** задайте (см. также **.env.example**):

```env
SMTP_HOST=smtp.example.com
SMTP_PORT=587
SMTP_USER=your-email@example.com
SMTP_PASSWORD=пароль_или_пароль_приложения
NOTIFICATION_EMAIL=получатель@example.com
```

**NOTIFICATION_EMAIL** — кому приходят напоминания и сгенерированные документы.

### 3. Ссылка на чат (в письме)

В **config/config.json** в секции **notifications** укажите URL веб-чата (если он доступен по ссылке):

```json
"notifications": {
  "chat_url": "http://localhost:8501",
  "days_before_event": 3
}
```

### 4. Vector Store (база знаний агента)

Чтобы сгенерированные документы попадали в базу знаний агента «Грантовый контролёр НейроПульс», в конфиге или в **config/.env** задайте **vector_store_id** (ID поискового индекса в Yandex AI Studio). Если не задан — документ всё равно сохраняется локально и отправляется на почту.

## Как это работает

1. **Скрипт напоминаний** (запуск вручную или по расписанию, например раз в день):
   ```bash
   python scripts/grant_reminders.py
   ```
   Находит события, до которых ровно 3 дня, и отправляет на **NOTIFICATION_EMAIL** письмо с текстом: «Через 3 дня — такое-то событие. Можем сформировать документ / сценарий / алгоритм. В чате напишите: Создать документы по событию «…»».

2. **Пользователь в веб-чате** пишет, например:  
   `Создать документы по событию «Промежуточный отчёт по этапу 1»`.

3. **Чат** находит событие в календаре и спрашивает: «Создать документы? Напишите Да или Нет».

4. При ответе **«Да»**:
   - агент генерирует текст документа (отчёт / сценарий / алгоритм);
   - создаётся файл Word в **data/generated_docs/**;
   - письмо с вложением отправляется на **NOTIFICATION_EMAIL**;
   - при заданном **vector_store_id** файл загружается в базу знаний агента.

---

## Расписание (Task Scheduler / cron)

Чтобы напоминания уходили автоматически за 3 дня до события, запускайте раз в день, например в 09:00:

- **Windows (Task Scheduler):** действие — запуск `python scripts/grant_reminders.py`, рабочая папка — корень проекта, интерпретатор — из `.venv`.
- **Linux/macOS (cron):** `0 9 * * * cd /path/to/neuropulse-grant-agent && .venv/bin/python scripts/grant_reminders.py`.
