# Подключение Яндекс Календаря к дашборду

Блок **«Ближайшие сроки по гранту»** в дашборде может показывать события из **Яндекс Календаря** по протоколу CalDAV.

## Шаг 1. Пароль приложения

1. Откройте [Пароли приложений](https://id.yandex.ru/security/app-passwords) в настройках Яндекс ID.
2. Нажмите **Создать пароль**.
3. Выберите тип **Календарь**.
4. Введите название (например, «Дашборд гранта») и создайте пароль.
5. Скопируйте выданный пароль — он показывается один раз.

## Шаг 2. Настройка в проекте

Задайте переменные окружения в `config/.env` (или в корневом `.env`):

```env
YANDEX_CALENDAR_USER=ваш_email@yandex.ru
YANDEX_CALENDAR_APP_PASSWORD=скопированный_пароль_приложения
```

Опционально — URL конкретного календаря (из веб-интерфейса [calendar.yandex.ru](https://calendar.yandex.ru): Календарь → ⋮ → Экспорт → скопировать из поля CalDAV):

```env
YANDEX_CALENDAR_URL=https://caldav.yandex.ru/...
```

Либо укажите те же параметры в **config/config.json** в секции `yandex_calendar`:

```json
"yandex_calendar": {
  "user": "ваш_email@yandex.ru",
  "app_password": "пароль_приложения",
  "calendar_url": ""
}
```

## Шаг 3. Дашборд

1. Запустите дашборд: `streamlit run src/dashboard/app.py`.
2. В блоке «Ближайшие сроки по гранту» нажмите **⚙️** (настройки).
3. Выберите источник **«Яндекс Календарь (CalDAV)»** и при необходимости измените период (дней вперёд).

События за выбранный период подгрузятся из вашего Яндекс Календаря.

## Зависимость

Установка: `pip install caldav` (уже добавлена в `requirements.txt`).

## Синхронизация с рабочим календарём

Яндекс Календарь поддерживает [синхронизацию по CalDAV](https://yandex.com/support/calendar/sync/sync-desktop.html) с сервером **caldav.yandex.ru**. Для дашборда используется тот же протокол: приложение подключается к календарю и читает события за указанный период.
