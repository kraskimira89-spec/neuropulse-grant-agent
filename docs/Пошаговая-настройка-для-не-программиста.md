# Пошаговая настройка (без опыта программирования)

Инструкция по шагам. После каждого шага — как проверить, что сделано верно.

---

## Шаг 1. Настроить почту (SMTP) для уведомлений и документов

**Зачем:** чтобы за 3 дня до события приходило письмо и чтобы готовые документы Word присылались на почту.

**Где:** папка проекта → папка **config** → файл **.env**  
(полный путь: `neuropulse-grant-agent\config\.env`)

### Что сделать

1. В Cursor слева откройте **Explorer** (иконка папок или Ctrl+Shift+E).
2. Раскройте папку **config**.
3. Щёлкните по файлу **.env** — он откроется в редакторе.
4. В **конец файла** (после строк с YANDEX_API_KEY и YANDEX_FOLDER_ID) добавьте такие строки. Подставьте **свои** данные вместо примеров:

```
SMTP_HOST=smtp.yandex.ru
SMTP_PORT=587
SMTP_USER=ваш_email@yandex.ru
SMTP_PASSWORD=пароль_приложения_яндекса
NOTIFICATION_EMAIL=куда_присылать_письма@example.com
```

**Пример для Яндекс-почты:**
- SMTP_HOST — `smtp.yandex.ru`
- SMTP_USER — ваш ящик, например `ivanov@yandex.ru`
- SMTP_PASSWORD — не обычный пароль, а **пароль приложения**: зайти на https://id.yandex.ru/security/app-passwords → создать пароль для «Почта» или «Другое» → вставить сюда.
- NOTIFICATION_EMAIL — адрес, на который должны приходить напоминания и документы (можно тот же ящик).

5. **Сохраните файл:** Ctrl+S или меню File → Save.

### Проверка шага 1

- В файле **config\.env** внизу есть строки SMTP_HOST, SMTP_PORT, SMTP_USER, SMTP_PASSWORD, NOTIFICATION_EMAIL.
- Нет лишних пробелов вокруг `=` и нет кавычек (пишем просто `SMTP_HOST=smtp.yandex.ru`).

**Напишите себе:** «Шаг 1 выполнен: в config\.env добавлены SMTP и NOTIFICATION_EMAIL» — и переходите к шагу 2.

## Шаг 2. Календарь событий гранта

**Зачем:** скрипт напоминаний смотрит даты событий из этого файла и за 3 дня шлёт письмо.

**Где:** папка **data** → файл **grant_calendar.json**  
(если его нет — сначала создаём из примера.)

### Что сделать

1. В **Explorer** (слева) откройте папку **data**.
2. Посмотрите, есть ли файл **grant_calendar.json**.
   - **Если есть** — щёлкните по нему, откройте и при необходимости поменяйте даты и названия событий (формат даты: `2025-04-15`).
   - **Если нет** — идите к пункту 3.
3. Щёлкните правой кнопкой по файлу **grant_calendar.example.json** → **Copy** (Копировать).
4. Щёлкните правой кнопкой по пустому месту в папке **data** → **Paste** (Вставить).
5. Переименуйте копию: имя должно быть **grant_calendar.json** (без .example). Для этого правый щелчок по новому файлу → **Rename** → введите `grant_calendar.json`.
6. Откройте **grant_calendar.json** и поправьте даты и названия под ваш грант. Пример одной записи:

```json
{
  "date": "2025-04-15",
  "title": "Промежуточный отчёт по этапу 1",
  "description": "Срок подачи отчёта",
  "document_type": "report"
}
```

7. **Сохраните:** Ctrl+S.

### Проверка шага 2

- В папке **data** есть файл **grant_calendar.json**.
- Внутри — массив событий с полями `date`, `title`, при желании `description` и `document_type`.

**Напишите себе:** «Шаг 2 выполнен: grant_calendar.json создан/обновлён» — и переходите к шагу 3.

## Шаг 3. Vector Store (база знаний агента) — по желанию

**Зачем:** если указать ID индекса, сгенерированные документы будут добавляться в базу знаний агента «Грантовый контролёр НейроПульс». Если не нужна загрузка в агента — шаг можно пропустить.

**Где:** либо **config\.env**, либо **config\config.json**.

### Что сделать (если нужна загрузка в агента)

1. Узнайте **vector_store_id** в Yandex AI Studio (раздел с поисковыми индексами / базой знаний агента).
2. Откройте **config\.env** и в конец добавьте строку (подставьте свой ID):

```
YANDEX_VECTOR_STORE_ID=ваш_идентификатор_индекса
```

3. Сохраните файл (Ctrl+S).

### Проверка шага 3

- Если вы добавили строку — в **config\.env** есть `YANDEX_VECTOR_STORE_ID=...`.
- Если шаг пропустили — ничего делать не нужно, документы всё равно будут сохраняться в папку и отправляться на почту.

**Напишите себе:** «Шаг 3 выполнен или пропущен» — и переходите к шагу 4.

## Шаг 4. Установить библиотеку для документов Word

**Зачем:** без неё генерация .docx не заработает.

**Где:** терминал в Cursor (внизу или через меню Terminal → New Terminal).

### Что сделать

1. Откройте **терминал** в Cursor: меню **Terminal** → **New Terminal** (или Ctrl+`).
2. Убедитесь, что вы в корне проекта. Должно быть что-то вроде:
   `...\neuropulse-grant-agent>`
   Если нет — введите и нажмите Enter:
   ```
   cd c:\Users\user\Documents\neuropulse-grant-agent
   ```
3. Включите виртуальное окружение (если ещё не включено). Введите и нажмите Enter:
   ```
   .venv\Scripts\Activate
   ```
   В начале строки должно появиться `(.venv)`.
4. Установите библиотеку. Введите и нажмите Enter:
   ```
   pip install python-docx
   ```
5. Дождитесь окончания. В конце должно быть сообщение вроде «Successfully installed python-docx-...».

### Проверка шага 4

- В терминале нет красного текста с ошибкой.
- Вывод содержит «Successfully installed python-docx».

**Напишите себе:** «Шаг 4 выполнен: python-docx установлен» — и переходите к шагу 5.

## Шаг 5. Запуск напоминаний вручную и по расписанию

### Запуск вручную (проверка)

1. Терминал: вы в корне проекта, включено `(.venv)`.
2. Введите и нажмите Enter:
   ```
   python scripts/grant_reminders.py
   ```
3. Должен появиться текст «Ближайшие сроки по гранту» и список событий. Если почта настроена и до какого-то события ровно 3 дня — уйдёт письмо.

### Проверка шага 5 (ручной запуск)

- Команда выполнилась без ошибки.
- В выводе виден список событий по календарю (или сообщение, что событий нет).

**Напишите себе:** «Шаг 5 проверен: скрипт напоминаний запускается».

### Напоминания каждый день (Task Scheduler в Windows)

Чтобы напоминания отправлялись автоматически раз в день:

1. Откройте **Планировщик заданий** Windows: Win+R → введите `taskschd.msc` → Enter.
2. Справа нажмите **Создать простую задачу**.
3. Имя, например: «Напоминания грант Нейропульс». Далее.
4. Триггер: **Ежедневно**. Далее.
5. Время — например 9:00. Далее.
6. Действие: **Запустить программу**. Далее.
7. В поле «Программа или сценарий» укажите полный путь к Python в вашем проекте, например:
   ```
   c:\Users\user\Documents\neuropulse-grant-agent\.venv\Scripts\python.exe
   ```
   (проверьте, что у вас такой же путь к папке проекта и есть папка `.venv`).
8. В поле «Добавить аргументы» введите:
   ```
   scripts/grant_reminders.py
   ```
9. В поле «Рабочая папка» введите:
   ```
   c:\Users\user\Documents\neuropulse-grant-agent
   ```
10. Готово → поставьте галочку «Открыть окно "Свойства"» → ОК. Задача будет запускаться каждый день в выбранное время.

### Проверка задачи в Планировщике

- В списке заданий есть «Напоминания грант Нейропульс» (или как вы назвали).
- В свойствах задачи указаны правильные путь к python.exe, аргумент и рабочая папка.
