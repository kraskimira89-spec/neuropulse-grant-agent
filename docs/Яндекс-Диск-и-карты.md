# Яндекс.Диск, контакты и карты в проекте

Кратко: как подключить и где используются.

---

## Папка проекта на Яндекс.Диске

Вся информация по гранту (документы, презентации, положение, заявки и т.п.) хранится в папке:

**[2025 Грант Губернатора](https://disk.yandex.ru/d/sqN9ph6tIGuc5g)** на Яндекс.Диске.

Сгенерированные агентом документы (отчёты, сценарии, алгоритмы) при включённой настройке загружаются в папку **«Файлы/ЦИП/ПРОЕКТЫ/2025 Грант Губернатора/Документы контролера»** на Диске (или в путь из `yandex_disk.upload_folder` / `YANDEX_DISK_UPLOAD_FOLDER`).

## Яндекс.Диск (выгрузка документов)

После генерации документа Word (по событию из календаря) копия загружается в папку проекта на Яндекс.Диске, если задан OAuth-токен.

### Настройка

1. Получите **OAuth-токен** один раз:
   - Зарегистрируйте приложение: [oauth.yandex.ru](https://oauth.yandex.ru/) → «Зарегистрировать новое приложение».
   - Тип: «Веб-сервисы», переадресация можно указать `https://oauth.yandex.ru/verification_code`.
   - Права: `cloud_api:disk.read`, `cloud_api:disk.write`.
   - Для личного использования: откройте в браузере  
     `https://oauth.yandex.ru/authorize?response_type=token&client_id=<ваш_Client_ID>`  
     и скопируйте токен из ответа.

2. В **config/.env** добавьте:
   ```env
   YANDEX_DISK_ACCESS_TOKEN=ваш_oauth_токен
   ```
   Папка загрузки по умолчанию: **Файлы/ЦИП/ПРОЕКТЫ/2025 Грант Губернатора/Документы контролера** (задаётся в **config/config.json** → `yandex_disk.upload_folder` или в .env → `YANDEX_DISK_UPLOAD_FOLDER`). Ссылка на папку проекта: `yandex_disk.project_folder_url`.

### Где используется

- При ответе «Да» в чате на «Создать документы по событию …»: документ сохраняется локально, отправляется на почту и при настроенном токене загружается в папку **Документы контролера** на Диске (путь из конфига).

## Контакты по гранту

Список команды и партнёров хранится в **data/grant_contacts.json** и отображается в дашборде в блоке «Контакты по гранту».

### Настройка

1. Скопируйте **data/grant_contacts.example.json** в **data/grant_contacts.json**.
2. Заполните массив: `name`, `role`, `email` для каждого контакта.

### Где используется

- Дашборд → блок «👥 Контакты по гранту»: таблица с именами, ролями и почтой.  
В дальнейшем этот список можно использовать для рассылки напоминаний или в отчётах.

## Адрес и ссылка на карты

В календаре событий (локальный **grant_calendar.json** или Яндекс.Календарь) у события можно указать место проведения.

### Локальный календарь (grant_calendar.json)

В каждом событии добавьте поле **address** (при необходимости):

```json
{
  "date": "2025-09-01",
  "title": "Начало этапа 2",
  "description": "Старт этапа",
  "address": "Челябинск, ул. Примерная, 1"
}
```

### Яндекс.Календарь (CalDAV)

В событии календаря укажите **Место** (LOCATION). Дашборд подхватит его как адрес.

### Где используется

- Дашборд → блок «Ближайшие сроки»: у события с заполненным адресом появляется кнопка **«Открыть в Яндекс.Картах»** (поиск по адресу, ключ API не нужен).

Опционально для геокодера (нормализация адресов) можно задать **YANDEX_MAPS_API_KEY** в .env (ключ из [developer.yandex.ru](https://developer.yandex.ru/)).
